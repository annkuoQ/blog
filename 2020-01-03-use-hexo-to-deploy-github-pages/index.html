用 Hexo 和 GitHub Pages 架部落格

AnnKuoQ

2020-01-03

資工 / 程設

為什麼要用 Hexo？

  • 因為中文資源比較多
  • 因為很多別人做好的主題可以直接用

執行環境

  • Win 10 與 Cmder

前置步驟

  1. 安裝 Git
  2. 安裝 Node.js
  3. 創建一個 GitHub Repository

安裝步驟

  1. 安裝 hexo
  2. cd 到桌面
  3. 建立網站所需檔案
  4. 現在的目錄結構變成這樣
  5. 進到資料夾中
  6. 安裝相關套件

建立網站

  1. 產生靜態網站
  2. 多了一個 public\ 資料夾出來，目錄結構如下
  3. 啟動本地服務器
  4. 前往 http://localhost:4000 顯示範例網站
  5. Ctrl+ C 停止服務器

建立文章

  1. 執行創建文章的指令

  2. 在 \source\_posts 底下新增了一個叫 post-title.md 的檔案
    打開檔案後，可以看到已經有些內容在裡面
    是剛剛打的標題和創建文章的時間

  3. 試著在下面打一些內容，並儲存

  4. 再次產生靜態網站

  5. 再次啟動服務器

  6. 前往 http://localhost:4000/test/2020/01/03/post-title/ 可以看到新文章出現

部署網站

  1. 安裝 Git 部署套件
  2. 開啟 _config.yml，編寫以下的值，並儲存檔案
  3. 部署到 GitHub
  4. 前往 https://annkuoQ.github.io/test 就可以看到自己的網站囉
    (開啟 GitHub Pages 功能需要一點時間，若頁面還沒出來，讓子彈飛一下)
